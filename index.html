☁️ Синхронизация

Подключено

Синхронизация через GitHub Gist — бесплатно, без карты. Нужен только GitHub-аккаунт.

Как получить GitHub Token (2 минуты)
  1. Зайди на github.com → Settings (аватарка → Settings)
  2. Developer settings → Personal access tokens → Tokens (classic)
  3. Generate new token → поставь галочку только на gist
  4. Скопируй токен и вставь выше
  5. На iPhone/iPad — введи тот же токен и тот же код синхронизации

Приложение создаст приватный Gist читалка-{код} и будет хранить там позиции и заметки.

✈️ Telegram — книги

Отправь книги своему Telegram-боту, и они появятся здесь для скачивания.

Как создать бота (1 минута)
  1. Открой Telegram → найди @BotFather
  2. Напиши /newbot → придумай имя
  3. BotFather пришлёт токен вида 1234567890:ABCdef...
  4. Вставь токен выше и нажми «Подключить»
  5. Открой своего бота → нажми Start → отправь ему книги как файлы
